Para capturar "_id" y el "date" de esta respuesta hay que contar en qué nivel de llave "{}" está. En este caso "_id" está dentro del segundo nivel de llave y "date" dentro del tercer nivel de llave, entonces se captura de la siguiente manera:

Para capturar "_id": se coloca primero el simbolo de dolar y seguido se agregan puntos según en qué nivel de llave esté la variable, para este caso está en el nivel 2, entonces se agregan 2 puntos y seguido se pone el nombre del atributo del que queremos guardar el valor, _id para este ejemplo.

$.._id

Para capturar "date": En este caso está en el nivel 3, entonces se agregan 3 puntos y seguido se pone el nombre del atributo del que queremos guardar el valor, date para este ejemplo.

$...date
